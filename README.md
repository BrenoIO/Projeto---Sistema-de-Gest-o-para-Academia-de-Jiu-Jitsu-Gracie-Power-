# Projeto---Sistema-de-Gest-o-para-Academia-de-Jiu-Jitsu-Gracie-Power-
A “Gracie Power” é uma academia de Jiu-Jitsu que oferece aulas para diferentes níveis técnicos. Seu público-alvo são academias e professores que desejam organizar melhor o controle de alunos e pagamentos. O sistema tem como propósito otimizar a gestão interna e apoiar decisões administrativas.
